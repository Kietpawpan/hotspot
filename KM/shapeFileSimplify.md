
# ลดขนาดไฟล์ GeoJSON

วิธีการดึงข้อมูลขอบเขตจาก OpenStreetMap (OSM) ผ่าน Overpass Turbo:

ไปที่เว็บไซต์ https://overpass-turbo.eu/

ลบคอลัมน์คำสั่งเดิมทางซ้ายมือทิ้งให้หมด แล้วก๊อปปี้คำสั่งนี้ไปวางแทนครับ (คำสั่งนี้จะดึงข้อมูลเฉพาะ 4 จังหวัดที่เราต้องการ)
```
[out:json][timeout:25];
// กรองพื้นที่ในประเทศไทย
area["ISO3166-1"="TH"][admin_level=2]->.searchArea;
// ดึงข้อมูล 4 จังหวัด
(
  relation["admin_level"="4"]["name:th"="จังหวัดนครราชสีมา"](area.searchArea);
  relation["admin_level"="4"]["name:th"="จังหวัดชัยภูมิ"](area.searchArea);
  relation["admin_level"="4"]["name:th"="จังหวัดบุรีรัมย์"](area.searchArea);
  relation["admin_level"="4"]["name:th"="จังหวัดสุรินทร์"](area.searchArea);
);
// แสดงผลเป็น GeoJSON
out geom;
```
กดปุ่ม Run ที่มุมซ้ายบน

รอสักครู่ ระบบจะดึงเส้นขอบเขตมาแสดงบนแผนที่ทางขวามือ (ถ้าเส้นขอบเขตดูโอเค ตรงกับแผนที่ฐาน แสดงว่าใช้ได้ครับ)

กดปุ่ม Export ด้านบน

เลือกดาวน์โหลดเป็น GeoJSON


ไฟล์ MAPS11_boundary_highres.geojson มีความละเอียดสูงมากทำให้โหลดผ่านเน็ตนานและคำนวณช้า หากไม่ได้ต้องการแสดงผลเส้นขอบเขตระดับซูมลึกมากๆ แนะนำให้นำไฟล์นี้ไปลดความละเอียดลงที่เว็บไซต์ [mapshaper.org](https://mapshaper.org/) (ใช้ฟังก์ชัน Simplify ลดทิ้งไปสัก 80%) จะทำให้หน้าเว็บโหลดเปิดมาเร็วขึ้นแบบก้าวกระโดด

การปรับลดขนาดไฟล์แผนที่ MAPS 11 ให้โหลดเร็วขึ้น โดยยังคงความแม่นยำของขอบเขตพื้นที่รับผิดชอบทั้ง 4 จังหวัด (นครราชสีมา ชัยภูมิ บุรีรัมย์ สุรินทร์) แนะนำให้ใช้การตั้งค่าใน Mapshaper ดังนี้ครับ

1. ในหน้าต่าง Simplification จากภาพให้ติ๊กเลือก prevent shape removal เพื่อป้องกันไม่ให้พื้นที่ขนาดเล็กถูกลบหายไป
2. เลือก Method เป็น Visvalingam / weighted area เพราะวิธีนี้จะรักษารูปทรงและแนวชายแดนได้ดีที่สุดโดยไม่ทำให้เกิดเส้นหยักแหลมผิดปกติ
3. กดปุ่ม Apply
4. เมื่อมีแถบเลื่อน (Slider) ปรากฏขึ้นด้านบนหน้าจอ ให้ปรับลดเปอร์เซ็นต์ลงมาที่ประมาณ 15% - 20%
5. ลองซูมตรวจสอบแนวเส้นขอบเขตของ สคพ.11 หากยังดูสมบูรณ์ดีให้ทำการ Export เป็นไฟล์รูปแบบ GeoJSON นำไปอัปโหลดใช้งานได้เลย

การลดรายละเอียดด้วยวิธีนี้ขอบเขตอาจเปลี่ยนไปจากเดิมเพียงระดับไม่กี่เมตรเท่านั้น ซึ่งมั่นใจได้ว่าจะไม่มีผลกระทบต่อความแม่นยำในการคัดกรองพิกัดของระบบ MAPS 11 แน่นอนครับ เพราะจุดความร้อนจากระบบดาวเทียม (อย่าง VIIRS) มีขนาดพิกเซลความกว้างถึง 375 เมตรซึ่งกว้างกว่าความคลาดเคลื่อนจากการลดขนาดไฟล์ไปมาก
