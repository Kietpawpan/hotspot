# Thailand - Subnational Administrative Boundaries
Get Notified

Thailand administrative level 0-3 boundaries (COD-AB) dataset version 01.
[https://data.humdata.org/dataset/cod-ab-tha](https://data.humdata.org/dataset/geoboundaries-admin-boundaries-for-thailand)

This dataset is structured into 3 levels:
- Admin 1: 77 Province
- Admin 2: 928 District
- Admin 3: 7425 Sub-district (Tambon)

Dates associated with this dataset:
- 30 October 2025: dataset reviewed for accuracy and completeness
- 22 January 2022: valid for use by the humanitarian community
- 01 January 2022: last edit to the dataset before publishing
- 01 January 2022: boundaries created by the source

Contributed by OCHA Regional Office for Asia and the Pacific (ROAP).
Quality assured, configured, and published by OCHA Field Information Services (FIS) and HDX.

Part of the dataset: Global Subnational Administrative Boundaries

รหัสในระบบฐานข้อมูลแผนที่สากล (เช่น HDX หรือ geoBoundaries) จะแบ่งระดับชั้นการปกครองตามตัวย่อ ADM (Administrative level) ดังนี้ครับ:

ADM0: ระดับประเทศ (Country)

ADM1: ระดับจังหวัด (Province)

ADM2: ระดับอำเภอ (District) 

ADM3: ระดับตำบล (Sub-district)
